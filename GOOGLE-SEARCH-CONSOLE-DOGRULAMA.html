Google Search Console Doğrulama Sayfası

Bu sayfa Google Search Console doğrulaması içindir.

Tarayıcıda şu adresi açın: https://mehmetcanbaykan.com/google-site-verification.html

Hızlı Adımlar

  1. Search Console'a git
  2. URL öneki olarak https://mehmetcanbaykan.com/ ekle
  3. "HTML etiketi" doğrulama yöntemini seç
  4. Google'ın verdiği meta tag'ı yukarıdaki content alanına yapıştır
  5. Bu dosyayı google-site-verification.html adıyla kaydet
  6. Vercel'e push'la → Search Console'da "Doğrula" tıkla
